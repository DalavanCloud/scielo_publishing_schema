.. _elemento-suffix:

<suffix>
^^^^^^^^

Aparece em
  :ref:`elemento-name`
 
Ocorre
  Zero ou mais vezes


Especifica sufixos do nome como as partículas "Neto", "Júnior", "Jr.", 
"Filho", "Sobrinho" etc.
 
Exemplo:

.. code-block:: xml
 
    ...
    <name>
        <surname>Santos</surname>
        <given-names>João da Silva</given-names>
        <suffix>Neto</suffix>
        ...
    </name>
    ...
