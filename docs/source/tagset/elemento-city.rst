.. _elemento-city:

<city>
======

Aparece em:

  :ref:`elemento-addr-line`

Ocorre:

  Zero ou mais vezes


Identifica a cidade da instituição vinculada ao autor, caso exista.

Exemplo:


.. code-block:: xml

    ...
    <addr-line>
        <city>São José do Rio Preto</city>
        <state>São Paulo</state>
        ...
    </addr-line>
    ...
