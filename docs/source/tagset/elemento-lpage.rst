.. _elemento-lpage:

<lpage>
-------

Aparece em
  :ref:`elemento-article-meta`, 
  :ref:`elemento-element-citation`
 
Ocorre
  Zero ou uma vez

 
Designa-se a paginação final do artigo. No caso de :term:`ahead-of-print`, 
a informação deve ser preenchida com ``00``.
 
Exemplo:

.. code-block:: xml
 
    ...
    <article-meta>
        ...
        <fpage>396</fpage>
        <lpage>452</lpage>
        ...
    </article-meta>
    ...
